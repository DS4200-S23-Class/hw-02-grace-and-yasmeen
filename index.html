Data Types

UFO

  • Row data type: Item
  • Column data type: Attribute
    • Attribute datetime: sequential quantitative
    • Attribute duration: sequential quantitative
    • Attribute city: nominal categorical
    • Attribute state: nominal categorical
    • Attribute country: nominal categorical
    • Attribute shape: nominal categorical

Airports

  • Row data type: Item
  • Column data type: Attribute
    • Attribute AIRPORT_NAME: categorical
  • Column data type LONGITUDE: position
  • Column data type LATITUDE: position

Google-Web

  • Row data type: item
  • Column data type: link

Task Abstraction

  • High Level Task: Consume - discover

    • Shows last 3 days' of data so user can look at where point aligns relative to other data.

  • Medium Level Task: Search - lookup

    • Visualization shows target data points for user to look at and the target and location are known.

  • Low Level Task: Query - identify

    • The data points are highlighted in red and are a single target to be identified.

  • Target: outliers

    • The highlighted data points lie away from the other data points indicating there is a different pattern compared to the rest of the data.

  • High Level Task: Consume - present

    • The visualization presents two statistics from two locations with clear and direct purpose.

  • Medium Level Task: Search - browse

    • The locations are known but there isn't a specified target for the user to look at.

  • Low Level Task: Query - compare

    • The visualization compares Seattle and Portland to measure temperature over time.

  • Target: similarity

    • There are two locations with the same temperature variable so the task is to find the similarities between the two.
